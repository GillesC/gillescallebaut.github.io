Java Tutorial 1 - Installeren van Eclipse Gepost op 28 juli 2014

Eclipse is een opensourceframework van de Eclipse Foundation voor software-ontwikkelomgevingen. De bekendste toepassing is het gebruik als IDE voor de programmeertaal Java.
Als je hier niets van begreep is dit niet erg, het enigste dat je moet weten is dat we Eclipse gebruiken om onze java programma's te schrijven en uit te voeren. Ook helpt dit naar het zoeken van 'fouten' en nog veel meer.

Installeren van Java SE Development Kit 8

Download de Java SE Development Kit.
Vergeet niet om de voorwaarden te accepteren. Download de gewenste versie, windows x64 (64-bit versie) of windows x86 (32-bit versie). Installeer nu de Development kit -door de stappen in de wizard te volgen-.

Installeren van Eclipse Luna

Hierna gaan we de laatste versie van Eclipse -momenteel Luna- downloaden. Wederom kies je de juiste versie, bij de meeste PC's zal dit de 64-bit versie zijn.
(De downloads bevinden zich rechts bovenaan, onder Download Links)

Als de download voltooid is, gaat u de zip-file uitpakken naar een logische locatie, bijvoorbeeld: c:\Program Files\Eclipse. Voor het uitpakken van een zip-file maakt u best gebruik van 7-zip. Nu kan u gewoon klikken op het .exe bestand om eclipse op te starten. Tijdens het opstarten zal er gevraagd worden naar de gewenste Workspace, kies hiervoor weer een logische locatie, bijvoorbeeld c:/Development/Java.

We kunnen eraan beginnen!
