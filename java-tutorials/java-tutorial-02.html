We overlopen nu de meestgebruikte primitieve data types in java, alsook Strings en Arrays -een datastructuur-.

Primitieve data types in Java

int
Het int datatype is een 32-bit signed two's complement integer.
In het Nederlands & vereenvoudigd: Een int is een natuurlijke getallen die binnen bepaalde grenzen valt qua grootte.
double
Het double data type is een dubbele-precisie 64-bit IEEE 754 floating point.
In het Nederlands & sterk vereenvoudigd: een double is een decimaal getal.
Eigenlijk is een double meer een benadering met een zeker precisie van een decimaal getal, meer hierover vind je hier.
boolean
Het boolean data type heeft enkel 2 waarden, true of false. Dit data type wordt gebruikt om condities aan te duiden.
char
Het char data type is een single 16-bit Unicode character, met andere woorden een karakter.
Info
Ik heb hier niet alle data types uitlegt. Er bestaat ook nog zoiets als short, long, float, ...
Als je op hierop klikt zal je naar een Wikipedia-pagina worden gebracht waar je meer kunt te weten komen over de primitieve data types.
Wat soms wordt gezien als een primitieve data type is de string maar dit behoort eigenlijk niet tot de primitieve data types. Strings zijn in java objecten, waar ik later op terug zal komen.
String
Het String object is een sequentie van karakters.

Data structuren in Java

Array
De meest gebruikte en gemakkelijkste datastructuur om mee te beginnen is de array.
Een array is niets meer dan een rij, die begint met een index 0. 

dataType[] arrayNaam;   //een array declareer je door eerst het soort datatype te vermelden die in de rij zit, 
                        //met daaropvolgend [] als symbool om aan te duiden dat het over een Array gaat
/*dit was de declaratie, nu de initialisatie*/
arrayNaam = new dataType[grootte]; //we hebben nu een array gecreëerd met een lengte 'grootte'

Arrays zullen uitvoerig besproken/uitgewerkt worden in volgende tutorials.

Voorbeelden

Algemeen

Voor het declareren van een variabele gebruiken we volgende syntax: DataTypeNaam naamVanVariabele;
Nu hebben we dus een variabele naamVanVariabele die behoort tot het data type DataTypeNaam.

Nu moeten we de variabele nog initialiseren, we moeten dus de variabele een eerste waarde geven: naamVanVariabele = waardeVariabele;
Nu hebben we heeft de variabele naamVanVariabele een waarde waardeVariabele gekregen.

Dit kunnen we nu allemaal samenvoegen tot 

DataTypeNaam naamVanVariabele = waardeVariabele;
