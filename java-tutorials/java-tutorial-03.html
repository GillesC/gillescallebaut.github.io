Kennismaking met data types, strings en arrays

Eerst gaan we dieper in op de char en array.

Char

Zoals ik al zei wordt een karakter aangeduid door ' '. We moeten duidelijk maken aan de computer dat we een karakter ingeven en geen code voor een karakter. De code voor een karakter wordt een ASCII-code genoemd. Een ASCII code is een natuurlijk getal dat wordt gegeven aan een karakter, deze getallen kan je terug vinden in een ASCII tabel. Zo kan men bijvoorbeeld zeggen dat een karakter 'a' kan voorgesteld worden door een ASCII-code van 97.

We verduidelijken dit door een voorbeeldje: 

             
public class EersteProgramma {
	public static void main(String[] args) {
		char letter = 'a';
		char letterAscii = 97;
		
		System.out.println("Dit is onze 'a': " +letter);
		System.out.println("Dit is onze 97: " +letterAscii); 
	}
}

We gaan de code hierboven eens verduidelijken.
char letter = 'a';, hier geven we aan dat het gaat over een karakter, 'a'. Door het gebruik van de enkele aanhalingstekens weet de computer dat het gaat over een karakter.
char letterAscii = 97;, hier geven we de ASCII-code door, namelijk 97.
We controleren nu of het inderdaad klopt door de waarde van karakter uit te printen naar het scherm, dit doen we door gebruik te maken van de code System.out.println();
Tussen de dubbele aanhalingstekens komt een String -wat tekst dus- met daarachter een + om aan te tonen dat de waarde van de karakters daar nog moet achter komen.

Even ter info
Je kan ook System.out.print() gebruiken.
Als je gebruik maakt van System.out.println() wordt er elke keer een nieuwe lijn gestart na het uitprinten, wat niet het geval is als je System.out.print() gebruikt.
Je kan ook handmatig ene nieuwe lijn invoeren door de String "\n", let op de backslash, dit is geengewone schuine streep.

Als we nu drukken op het icoontje bovenaan links, om het programma uit te voeren dan verkrijgen we het volgende:
Dit is onze 'a': a
Dit is onze 97: a

Dus we kunnen een karakter voorstellen door zijn karakter-waarde (ASCII-code) of door de karakter-voorstelling zelf.


Array

Een array is -zoals al eerder gezegd- een rij van verschillende objecten/data types,... Een rij wordt aangeduid door [] en begint met een index nul. 

int[] rijGetallen = new int[10];
In dit voorbeeld hebben we dus een lege array met plaats voor 10 elementen van het type int. We kunnen nu de rij opvullen: 
rijGetallen[0] = 1;
rijGetallen[1] = 2;
rijGetallen[2] = 3;
//enzovoort...
rijGetallen[9] = 10;
We kunnen dus waarden in de rij steken door te werken met de index van de rij.
Ook kan je gebruik maken van 
int[] rijGetallen = {1,2,...,9,10};
Via de index van de rij kan je zo weer aan de waarde geraken die je er oorspronkelijk hebt in gestoken. 
public class EersteProgramma {
	public static void main(String[] args) {
		int[] rijGetallen = new int[2];
		rijGetallen[0] = 1;
		rijGetallen[1] = 2;
		System.out.println(rijGetallen[1]);
	}
}
Als we dit programma nu uitvoeren, dan bekomen we 2, wat dus de waarde was die zat op de 2de plek in de rij (maar met index 1!).
