Java Tutorial 5 - if/else statement & lussen Gepost op 16 aug. 2014

Programmeurs zijn in weze zeer luie mensen, als een computer het werk voor ons kan doen dan laten we hem werken i.p.v. onszelf moe te maken. Daarom gebruiken we -onder andere- lussen. Lussen gebruiken we om code een aantal keer te laten doorlopen.

if/else statements gebruiken we om bepaalde code te doorlopen enkel als onze conditie juist trueof fout false is.

if/else statement

Het is vaak handig om enkel een stuk code uit te voeren wanneer een conditie waar is of als het juist fout is.

Als voorbeeld, vragen we ons af hoe het weer momenteel is. Als het goed weer is dan is het misschien best om een korte broek aan te trekken, als het slecht weer is dan is het misschien beter om een paraplu mee te nemen.
Om dit naar code te 'vertalen' maken we dus gebruik van het if/else statement. In de comments wordt er weergegeven hoe het eruit ziet in zijn algemene vorm. 

             
public class WatAanTeDoen {
	public static void main(String[] args) {
		String weer = "regen";
		
		/*
		 * if(conditie){
		 * 		//doe iets
		 * } else {
		 * 		//doe iets anders
		 * }
		 */
		
		if(weer=="regen"){
			System.out.println("Het is slecht weer vandaag, doe maar een paraplu mee!");
		} else {
			System.out.println("Het ziet er goed weer uit vandaag, doe maar een korte broek aan.");
		}
	
	}
}

Toelichting van de code

We maken via String weer = "regen"; een string aan met de naam weer en de waarde "regen". Achter if -tussen de haakjes- kijken we of deze statment juist of fout is. Dit doen we door te 'vragen', weer == "regen" heeft de string weer een waarde gelijk aan "regen"?
We gebruiken hier 2 maal het gelijkheidsteken om te kijken of de waarde van weer gelijk is aan "regen", moesten we gebruik maken van een enkele '=' dan zouden we zeggen dat de string weer een waarde van "regen" zou krijgen.

We kunnen het if/else statement verder uitbreiden, door deze meerdere keren te herhalen. 

public class WatAanTeDoen {
	public static void main(String[] args) {
		String weer = "zonnig";
		
		/*
		 * if(conditie){
		 * 		//doe iets
		 * } else if(conditie2){
		 * 		//doe iets anders
		 * }else{
		 * 			//als de condities hierboven allemaal niet voldoen zijn
		 *			//doe dan maar nog iets anders
		 * }
		 */
		
		if(weer=="regen"){
			System.out.println("Het is slecht weer vandaag, doe maar een paraplu mee!");
		} else if(weer=="mistig") {
			System.out.println("Het is momenteel mistig, blijf maar lekker binnen.");
			} else if(weer=="zonnig"){
			System.out.println("Het ziet er goed weer uit vandaag, doe maar een korte broek aan.");
			} else{
				System.out.println("Het is vandaag niet regenachtig, mistig of zonning, "
						+ "dat is het enigste dat ik zeker weet."
						+ "Dus kijk zelf maar eens buiten, eh!");
			}
	}
}

Toelichting van de code

De laatste else statement in de code wordt uitgevoerd als er geen van de bovenstaande condities kloppen.
De plusjes die je ziet bij de laatste string tussen System.out.println() is om de leesbaarheid van de code te verbeteren, zo wordt de code niet te breed. We zetten dus een plus om verschillende strings met elkaar te 'verbinden'.

Opmerking
Het is niet de bedoeling om die if/else statement teveel te herhalen zoals hierboven. Daarvoor bestaat er zoiets als een switch statement, die hieronder wordt besproken.

Lussen

Als we een code verschillende keren willen uitvoeren maken we gebruik van lussen.
De lussen die we gaan besproken: de for-lus, de while-lus en de do-while-lus. We gebruiken deze lussen door elkaar, maar de keuze van welke lus we gebruiken is niet willekeurig.

Er bestaat ook nog zo iets als een for-each lus, maar deze zullen we bespreken als we het gaan hebben over lists.

De for-lus

Gebruik: Als we weten hoeveel keer we de lus moeten doorlopen -bijvoorbeeld bij het opvullen van een array- maken we gebruik van een for-lus. 

public class ForLusVoorbeeld {
	public static void main(String[] args) {
		int rij[] = new int[10];
		
		//we willen nu de rij opvullen van 0 tot 9
		//we weten dus dat we de lus 10 keer moeten doorlopen
		
		/*
		 * for (initialisatie; terminatie; toename/afname) {
		 * 		//statement
		 * }
		 */
			     
		for(int i=0;i<10;i++){
                    rij[i]=i;
                    system.out.println("de waarde van de rij "+rij[i]);
                }
        }
}

Toelichting van de code

We maken eerst een rij array aan van 10 ints.
De for-lus werkt zoals volgt,

initialisatie int i=0;
Dit wordt eenmaal uitgevoerd wanneer de lus wordt uitgevoerd
terminatie i<10;
Als de conditie false is dan wordt de lus gestopt.
toename/afname i++;
De waarde van i wordt met 1 vermeerderd, elke keer dat de lus wordt doorlopen. Als we gebruik maken van i--; dan wordt i met 1 verminderd.
Dus de lus wordt 10 keer doorlopen. Beginnend met i=0 en stopt bij i=10, waarbij de lus dus niet meer doorlopen wordt.

De while-lus

Gebruik: Als we niet weten hoeveel keer we de lus moeten doorlopen én wanneer we niet weten of de lus wel moet doorlopen worden, dan maken we gebruik van een while-lus.

Een while lus kijkt of een conditie waar is -true- en dan doorloopt hij de code die zich bevindt in de while-lus.
Voor we beginnen met een voorbeeld van een while lus gaan we users inputs verwerken. We maken een simpele input scanner aan via Scanner input = new Scanner(System.in);. Hiervoor moet je wel de Java.util.Scanner bibliotheek importeren -zie mijn programam hieronder-. De users input -via het keyboard- worden gelezen door de volgende code, input.nextInt();. Zoals je kan zien aan de code gaan we int's inlezen. We kunnen ook strings inlezen via String zin = input.nextLine();. 

import java.util.Scanner; //importeren van de scanner
public class WhileVoorbeeld {
	public static void main(String[] args) {
		int n; //declaratie n
		 
	    Scanner input = new Scanner(System.in); //scanner
	    
	    System.out.println("Geef een getal in met uitzondering van 0");
	    n = input.nextInt(); //lees het getal in
	 
	    /* zolang de user een getal ingeeft verschillend van 0
	     * wordt de lus doorlopen
	     */
	    while (n != 0) {
	      System.out.println("Uw getal: " + n);
	      System.out.println("Geef een getal in en stop door 0 in te typen");
	      n = input.nextInt(); //lees het getal in
	   }
	    input.close(); //het sluiten van de scanner is niet noodzakelijk
	  }
}

De do-while lus

Gebruik: Als we te maken hebben met een lus dat zeker éénmaal moet worden doorlopen én het aantal keer dat de lus moet doorlopen worden is niet gekend, dan maken we gebruik van een do-while lus.

We nemen hetzelfde voorbeeld als hierboven maar dan met een do-while lus. Dan bekomen we dit: 

import java.util.Scanner; //importeren van de scanner
public class DoWhileVoorbeeld {
	public static void main(String[] args) {
		int n; //declaratie n
		 
	    Scanner input = new Scanner(System.in); //scanner
	    
	    System.out.println("Geef een getal in met uitzondering van 0");
	    n = input.nextInt(); //lees het getal in
	 
	    /* zolang de user een getal ingeeft verschillend van 0
	     * wordt de lus doorlopen
	     */
	    do{
	    	System.out.println("Uw getal: " + n);
		      System.out.println("Geef een getal in en stop door 0 in te typen");
		      n = input.nextInt(); //lees het getal in
	    }
	    while (n != 0);
	    input.close(); //het sluiten van de scanner is niet noodzakelijk
	  }
}

Als we nu dit programma laten uitvoeren en we voeren '0' in dan stopt de code niet, in tegenstelling tot het voorbeeld met de while-lus. De code in de do-while lus wordt dus eerst doorlopen om dan pas nadien te kijken of er aan de conditie voldaan is.

Switch statement

Gebruik: In plaats van lange if... else if statements te gebruiken, kiezen we voor een switch-statement. De switch kijkt naar je variabele en vergelijkt deze met een range van cases.
Als de geteste variabele overeenkomt met een case dan voert hij de code uit die behoort bij die case. 

import java.util.Scanner; //importeren van de scanner
public class SwitchVoorbeeld {
    public static void main(String[] args) {
        int n; //declaratie n
         
        Scanner input = new Scanner(System.in); //scanner
        
        System.out.println("Geef een getal tussen 0 en 4");
        n = input.nextInt(); //lees het getal in
     

        switch(n){ //n is het ingegeven getal
            case 1: System.out.println("U heeft 1 ingedrukt, proficiat"); break; 
            case 2: System.out.println("U heeft de vierkantswortel van 4 ingedrukt"); break; 
            case 3: System.out.println("U heeft 2 tot de 2de macht min één ingedrukt"); break; 
            default: System.out.println("U heeft niet goed geluisterd, een getal TUSSEN 0 en 4, moest je invullen!");
        }
        input.close(); //het sluiten van de scanner is niet noodzakelijk
      }
}

Toelichting van de code

Algemeen:

switch(variabele) waarbij we een variabele gaan 'controleren'.
case waarde:, als de variabele een waarde heeft gelijk aan deze van onze case dan wordt de code uitgevoerd die achter de ':' staat.
Om te weten wat de break; daar doet, moet je eens het getal 1 ingeven en de breaks overal weghalen. Je zult zien dat dan alle code gaat uitgevoerd worden, in plaats van enkel de code die behoort bij case 1:. break; wordt dus gebruikt om los te komen uit de case.
Een goeie gewoonte is ook altijd een default case aan te maken zodat als de variabele geen match heeft dat dan de code na de default wordt uitgevoerd.

Opmerking
Als er meerder gevallen -cases- zijn die éénzelfde code moet uitvoeren kan je gewoon deze cases achter elkaar schrijven. Bijvoorbeeld om te kijken of de gebruiker te jong is om bier te drinken.
case 1: ... case 14: case 15: System.out.println("Nog even wachten voor je eerste pint!");break; die '...' is geen code maar is om de code niet nodeloos lang te maken voor jullie. Hopelijk begrijpen jullie dat we telkens case X moeten invullen achter elkaar, met X die gaat van 1 t.e.m. 15.
