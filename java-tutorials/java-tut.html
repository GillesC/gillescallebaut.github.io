Java, een programmeertaal voor iedereen

Java is een computer programmeertaal dat klasse-gebaseerd, object-georienteerd is.
Met als bedoeling "write once, run anywhere" (WORA), wat betekent dat de code op eender welk platform kan draaien d.m.v. een Java virtual machine (JVM).
Java is een van de meeste populaire programmeertalen met over 9 miljoen developers.
De taal is gebaseerd op de syntax van C en C++.
